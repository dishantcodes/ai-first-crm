# 🚀 AI-First CRM

AI-powered Healthcare Professional Relationship Management System.

## 📌 Overview

AI-First CRM is a full-stack application that helps manage Healthcare Professionals (HCPs), track interactions, and generate AI-powered summaries and next actions using Large Language Models.

## 🛠 Tech Stack

### Frontend
- React.js
- JavaScript
- Axios
- CSS

### Backend
- FastAPI
- Python
- SQLAlchemy
- PostgreSQL

### AI Layer
- LangGraph
- Groq LLM
- AI Workflow Automation

## ✨ Features

### Healthcare Professional Management
- Add HCP details
- View healthcare professionals
- Manage doctor information

### Interaction Management
- Record doctor interactions
- Store notes and history
- Track follow-ups

### AI Features
- Generate interaction summaries
- Suggest next actions
- Chat with AI CRM assistant
